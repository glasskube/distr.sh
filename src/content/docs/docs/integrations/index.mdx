Distr integrates seamlessly with your development workflow and existing tools. Explore our APIs, SDKs, and platform integrations to automate your software distribution pipeline.

## API & SDKs

Access Distr programmatically using our REST API and client libraries:

- **[API Reference](https://app.distr.sh/docs)** - Complete REST API documentation for managing applications, deployments, and customers
- **[JavaScript/TypeScript SDK](/docs/integrations/sdk/)** - Official SDK for Node.js and TypeScript applications
- **[Personal Access Tokens](/docs/integrations/personal-access-token/)** - Generate API tokens for authentication

## CI/CD Integrations

Automate your deployment pipeline with native integrations:

- **[GitHub Actions](/docs/integrations/gh-action/)** - Automatically create versions and trigger deployments from your GitHub workflows

## AI & Development Tools

- **[Model Context Protocol (MCP)](/docs/integrations/mcp/)** - Integrate Distr with AI assistants like Claude for natural language deployment management
